Methodic
========

Methodic provides a GUI for triggering functions defined in a game object's
components. This can be used to manually trigger events during game play and to
modify game objects in the editor.

Open the Methodic window by selecting the Window > Methodic menu or using
hotkeys shift-command-m (Mac) / shift-control-m (Windows).
